# G-Code Generator Tools - Pen Plotter

A collection of web-based tools for generating G-code for pen plotters and CNC machines.

## Tools

- **Image to Points by Layers**: Convert images to G-code by generating points organized in brightness layers
- **Image Manipulator**: Preview and adjust images for plotting (no authentication required)
- **Radial Pattern Generator**: Generate radial patterns with customizable circles and Z-depth variations
- **Random Path Generator**: Create random tool paths with adjustable parameters
